# Code of Conduct

## Our Standard

Please keep discussions respectful, direct, and constructive.

Examples of expected behavior:

- Be specific about bugs and proposed fixes
- Respect different experience levels
- Keep feedback focused on the work

Examples of unacceptable behavior:

- Personal attacks
- Harassment
- Derailing technical discussion

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this code of conduct.
